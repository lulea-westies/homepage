# Luleå Westies

Welcome to the official website repository for **Luleå Westies** — the West Coast Swing dance community in Luleå, Sweden.

## About

Luleå Westies is a community for West Coast Swing dancers in Luleå. We organize classes, social dances, and events, and welcome both new and experienced dancers to be part of our community. West Coast Swing is a partner dance known for its improvisational style, musicality, and welcoming community.

## About This Repository

This repo contains the source code for the Luleå Westies website, built with [Hugo](https://gohugo.io/) — a fast and flexible static site generator. The site is hosted on GitHub Pages and automatically deployed when changes are pushed to the `main` branch.
